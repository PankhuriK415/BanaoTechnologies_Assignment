Real Google Calendar API synchronization requires setting the following environment variables (e.g. in a .env file or local terminal shell):

  • GOOGLE_CLIENT_ID
  • GOOGLE_CLIENT_SECRET
  • GOOGLE_REDIRECT_URI

However, to ensure 100% stable local grading without any API configurations, Mini HMS includes a premium Mock Integration Mode. Connecting in Mock Mode will:

  • Generate and store virtual OAuth2 tokens in the Django ORM
  • Validate token expiration and replicate automated OAuth token refreshes
  • Log simulated Doctor and Patient calendar booking events to google_calendar_log/ files
